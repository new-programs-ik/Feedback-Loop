11
Deep dive — Uplevel: video &amp; transcript
When a class is flagged, we need its recording and 4-hour transcript. We plan for both possibilities:
    If Uplevel has an API
The system pulls the recording and transcript automatically, by class / session ID.

Fully hands-off — the moment a class is flagged, the transcript is fetched.
    If it doesn’t (fallback)
Take the recording from where it’s stored (e.g. the Zoom cloud), transcribe it, and drop the transcript in — or the PM uploads it.

A few extra minutes, but the result is identical.
Either way, the rest of Flow 2 runs the same. This is the one connection we confirm early.

Deep dive — Metabase &amp; the cohort sheet
    Metabase  (ratings)
Connects through Metabase’s API using a saved question that returns per-session ratings.

Checked on a schedule. 4.5 and above is good; anything below is flagged into Flow 2.
    Cohort planning sheet
Your existing workbook — one sheet per cohort, with week, live-class date, topic and instructor.

The system reads schedule, topics and instructor names, and writes assignments back. Moves to a database later.
We start on the tools you already use — then migrate the data into a proper database in Phase 3.

2
The problem today
Almost everything around a weekly class is done by hand:
    Availability
Going to each instructor’s Discord channel to ask if they’re free — manually, every week.
    Materials
Digging through Google Drive to find and share the right folder.
    Ratings
Checking Metabase by hand to see how each class scored.
    Feedback
Watching recordings and writing feedback when a rating is low.
    Fragile sheets
A schedule kept in spreadsheets that break — #REF! errors and dead links.

3
What we’re building
One screen — the Console — with small automated workers behind it. You stay in control of every decision that matters.
    Staffs each week
Proposes instructors and gets their confirmation on Discord.
    Shares materials
Sends the right Instructor Copy folder automatically.
    Tracks ratings
Reads Metabase and flags anything below 4.5.
    Drafts feedback
Reads the transcript and writes feedback for your approval.
    Keeps you in control
Proposes and drafts; you approve the calls that matter.
    Flexible by design
The wiring is editable data, not buried in code.

What happens every week

8
The safety net — the Control Plane
    Audit log
Every action is written down with a timestamp and the reason. Nothing happens invisibly.
    Undo / rollback
Wrong folder shared or wrong invite sent? Reverse it with one click.
    Lessons &amp; rules
Correct it once, and your correction becomes a plain-English rule it follows.
    Does the AI rewire itself when you correct it?
No — and you wouldn’t want that. Models don’t retrain on the fly. Instead, your correction becomes a written rule you can read, edit and delete — and the system asks you to confirm it before it takes effect.
